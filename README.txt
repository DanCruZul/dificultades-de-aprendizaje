Descripción de la actividad:

Desarrollo de trabajo para niños con dificultades de
aprendizaje.

Esta labor consiste en hacer una página web donde se
encuentren 10 ejercicios o fichas de trabajo para cada
dificultad y direcciones web donde encontremos en
video ejercicios para cada dificultad de aprendizaje.

Cada módulo tendrá un valor de 4 horas

Requisitos para aprobación de cada modulo

* 10 fichas de trabajo para cada dificultad

* 5 direcciones de videos que contengan ejercicios
para cada dificultad

* Entregar la dirección o enlace de la página web al
Rector de la Fundación por WhatsApp.

Temas:

* Tema 1:Actividades para Dislexia

* Tema 2: Actividades para Disgrafía

* Temas: Actividades para discalculia

* Tema: Actividades para atención dispersa

* Tema5: Actividades para concentración

* Tema 6: actividades para memoria a corto plazo

* Tema7: Actividades para TDA déficit de atención
con hiperactividad

* Tema 8: Actividades para inteligencia emocional

* Tema 9: actividades para niñQss con altas
capacidades cognitivas

* Tema 10: Actividades para niños con autismo y
asperger
